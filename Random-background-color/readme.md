# 🎨 Random Background Color Changer

A simple JavaScript project that randomly changes the background color every second. Control the effect with **Start** and **Stop** buttons.

## 🚀 Features
- **Random Color Generation**: Changes the background to a random hex color every second.
- **Start/Stop Controls**: Use buttons to toggle the color change.
